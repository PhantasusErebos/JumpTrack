**JumpTrack** is a modern, offline-first web application designed to help you track and manage your **Jumpchain** adventures. Keep your builds organized, manage your CP budget, and document your journey across the multiverse with a premium, user-friendly interface.

## Getting Started

1.  **Open the App**: Simply open `index.html` in any modern web browser. No installation or server required.
2.  **Create a Jump**: Click the **+** button in the sidebar to start your first Jump.
3.  **Add Entries**: Use the input fields to add Perks, Items, and Drawbacks.
4.  **Check Profile**: Click the "Profile" button to see a global view of all your accumulated powers and items.

## Project Structure

- `index.html`: The main application entry point.
- `style.css`: Core styles and themes.
- `script.js`: Application logic (state management, UI rendering, data persistence).
- `print.html` & `print.css`: Dedicated print view logic and styling.
- `template_data.json`: A sample JSON file to test import functionality.

## Usage Tips

- **Shortcuts**: Use the "Print Build" button to quickly see a summary of your current progress.
- **Backup**: Regularly export your data using the "Export JSON" button to prevent data loss if you clear your browser cache.
